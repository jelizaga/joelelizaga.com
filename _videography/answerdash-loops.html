Inline Website Loops

Loops are becoming a common way to provide documentation and market software product features on the web. Visitors to your site can watch a human perform operations or access features in semi-real time... over and over and over again, so they don't miss a thing, and in digestible pieces!

The loops I've done for AnswerDash are crafted to "end how they start;" the end-point is the initialization of the loop.

The artist of the illustrations is unknown to me; AnswerDash hired a woman in the Slovakia (if my memory serves correct) to create these assets via Upwork as a freelance job. I wish I could credit the designer and link to her portfolio.

Basic AnswerDash deflection: The user opens the tab, finds Q&A determined by the machine learning algorithm to be relevant, and finds their answer.

Contextual Q&A: AnswerDash's Q&A is different on every page of your site.

Search: AnswerDash's search feature deflects further customer support tickets by allowing users to search Q&A for their questions.

Ticket deflection: When the user moves to submit a ticket, AnswerDash Pro presents Q&A relevant to their query using natural language processing.

AnswerBot is AnswerDash's Predictive Q&A Engine implemented on Facebook Messenger, yielding instant response times with automated self-service support.

AnswerDash Pro has mobile browser support. The engine functions the same on a mobile device, with a compacted appearance optimal for smaller screens.
